# Portfolio Template

A modern, responsive portfolio website template built with HTML, CSS, and JavaScript.

## Project Structure

```
portfolio-template/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Custom styles and animations
├── js/
│   └── script.js       # JavaScript functionality
├── assets/
│   └── images/         # Image assets directory
├── .gitignore          # Git ignore rules
└── README.md           # Project documentation
```

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, modern interface with smooth animations
- **Project Showcase**: Interactive project cards with modal details
- **Smooth Scrolling**: Smooth navigation between sections
- **Mobile Menu**: Animated mobile navigation menu
- **Device Mockups**: Responsive device mockups for project displays

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styles, animations, and responsive design
- **JavaScript**: Interactive functionality and animations
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **Font Awesome**: Icon library (via CDN)
- **Google Fonts**: Inter font family

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content, images, and styling to match your needs

## Customization

### Adding Images

Place your project images in the `assets/images/` directory and update the image paths in `index.html`.

### Modifying Styles

Edit `css/styles.css` to customize colors, fonts, spacing, and animations.

### Updating Content

Edit `index.html` to update:
- Personal information
- Project details
- Contact information
- Social media links

### JavaScript Functionality

Modify `js/script.js` to add or customize interactive features.
